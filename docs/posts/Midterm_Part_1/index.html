Demonstration

Demonstrate your ability to use these basic programming concepts in R.

Question 1:

# Create a character object
my_string <- "Hi Professor Crump!"
print(my_string)
[1] "Hi Professor Crump!"
# Create a numeric object
my_age <- 23
print(my_age)
[1] 23
# Create an integer object
number_one_prof <- 1L
print(number_one_prof)
[1] 1
# Create a logical object
the_best_prof <- TRUE
print(the_best_prof)
[1] TRUE
# Create a data frame
my_dataframe <- data.frame(x = c(1, 2, 3), y = c("a", "b", "c"))
print(my_dataframe)
  x y
1 1 a
2 2 b
3 3 c
# Create a list
my_list <- list(a = 1, b = "love RStudio", c = TRUE)
print(my_list)
$a
[1] 1

$b
[1] "love RStudio"

$c
[1] TRUE
# Indexing objects
# Index a character object
print(my_string[1])
[1] "Hi Professor Crump!"
# Change an element of a list
my_list[[2]] <- "Sike"
print(my_list)
$a
[1] 1

$b
[1] "Sike"

$c
[1] TRUE

Question 3:

# Make a loop do something 5 times
# i is the iterator
# 1:5 creates a vector with 5 numbers in it, 1, 2, 3, 4, 5
# the loop will run 5 times, because there are five things to assign to i
for(i in 1:5) {
  print("Psych Rules")
}
[1] "Psych Rules"
[1] "Psych Rules"
[1] "Psych Rules"
[1] "Psych Rules"
[1] "Psych Rules"
